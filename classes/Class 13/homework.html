1. UTF-8 encoding ke baare main thora prhna hai

Computer text ko pehle numbers mein convert karta hai (Unicode ke codes ki madad se), aur UTF-8 un numbers ko binary format mein badal kar store karta hai, taake computer unhe asaani se samajh sake. UTF-8 ka sabse bada fayda yeh hai ke yeh duniya ki har language ke characters ko support karta hai, jaise alphabets (English, Urdu, etc.), emojis, aur special symbols. Is wajah se hum ek hi document mein alag-alag languages aur emojis ko ek saath likh sakte hain. Web pages mein agar UTF-8 encoding define na ki jaye, toh kuch text ya symbols sahi display nahi hote. Isliye HTML pages mein UTF-8 encoding use karna zaroori hai, taake saara content properly dikhai de.

2. meta tags ke different aur bhi attributes dekhne hain

meta name="application-name" content="value" specifies the name of the Web application that the page represents. apni website ka naam is main put kr skte hain

meta http-equiv="refresh" content="10" web page ko har 10 seconds baad automatically refresh kar deta hai. ye ek basic aur purani technique hai jo ab modern use cases ke liye recommended nahi hai. Agar aapko data dynamically update karna hai toh AJAX ya WebSocket jaisi approaches ka use karein. Refresh karte rehne se unnecessary load aur poor user experience ka risk hota hai.
