Values of target attribute like _self, _parent, _top kia kaam krta hai



Click me to open in the named iframe

_self

Opens the linked document in the same frame as it was clicked aur ye value by default hoti hai.

_parent

Opens the linked document in the parent frame

_top

Opens the linked document in the full body of the window

framename

Opens the linked document in the named iframe

_parent Sirf ek level upar ke parent frame mein open karta hai.
_top Sabse top-level frame ka structure tod kar, browser window mein open karta hai.
