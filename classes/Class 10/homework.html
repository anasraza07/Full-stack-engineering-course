2. Input type tel ke kch patterns ke baare main bataana hai

Regular Expression (Pattern) Banane Ka Tareeqa:

  • \d: Ek digit (0-9) ke liye.
  • [0-9]: Sirf digits ko allow karega.
  • {n}: Ek fixed number of characters ke liye.
    Example: {3} ka matlab hai exactly 3 characters hone chahiye.
  • | : "OR" ke liye.
  • () : Grouping ke liye.
  • - : Dash ko explicitly allow karne ke liye.

Required Attributes for User Clarity:

placeholder: Yeh user ko correct format dikhane ke liye hai.

title: Agar input incorrect ho, toh browser error message ke sath yeh title bhi show karega.
