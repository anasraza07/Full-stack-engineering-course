1. How can we use anchor tag for bookmark in website?

HTML links can be used to create bookmarks, so that readers can jump to specific parts of a web page.

When the link is clicked, the page will scroll down or up to the location with the bookmark.

Pehle ham id attribute ka use krke bookmark create krte hain aur phr anchor tag ke href attribute main us id ki value dete hain kch is tarah:

        
    <h3 id="C4">Chapter 4</h3>
    <a href="#C4">Jump to Chapter 4</a>

Agr another page ke bookmark ka link dena ho to kch is tarah:

    
        <a href="another-page.html#C4">Jump to Chapter 4</a>

6. What is inline-block element?

An inline-block element in HTML is a type of element that combines the behavior of both inline and block elements, meaning it can flow with other text on the same line like an inline element, but also allows you to set width and height properties like a block element, enabling more flexible layout control within a webpage. And ham inline-block elements ki all sides ko margin & padding bhi de skte hain

7. <img> is which type of element?

<img> is an inline and empty element, which means that it doesn't start on a new line and doesn't take a closing tag.
