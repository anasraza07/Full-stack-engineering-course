Seven (7) layers OSI model

Ye eik model hai jo network communication ko 7 steps main divide krta hai

Think of it like postal service: hr layer ka eik specific kaam hota hai

Layer Kaam
1. Physical Wires, cables, signals: Data ka hardware level transmission
2. Data link Data ko frame main torna aur error fix krna
3. Network Data ko eik system se doosre system tk route krna using IP address
4. Transport Data ko reliable banaata hai (TCP/IP protocols)
5. Session Eik session create krna (connection manage krna)
6. Presentation Data ko format krna (encryption, compression)
7. Application End-user applications (jaise browsers, emails)
