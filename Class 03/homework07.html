TCP/IP kia hota hai?

TCP/IP eik protocol hai jo internet communication ke rules set krta hai

TCP (Transmission Control Protocol): reliable delivery ka dhiyan rkhta hai. Data ko chotay packets main torta hai aur ensure krta hai ke sb packets sahi sequence main reveiver tk pohanchen

IP (Internet Protocol): packets ko address krta hai aur unka route decide krta hai

Example: sender se receiver tk ka shortest route
