What is IPv4 & IPv6?

IPv4 aur IPv6 ka kaam hai devices ko unique address dena, data routing karna, aur ensure karna ki internet aur network smoothly chale. IPv6 zyada advanced hai, aur aane wale time mein IPv4 ko replace kar dega.

Data Routing (Addressing)

Jab tum kisi website ko open karte ho (jaise www.google.com), tumhara browser request karta hai data ke liye. IP address ye ensure karta hai ki request sahi server tak jaye aur data wapas tumhare device par aaye.

IPv4 home networks, small businesses, aur basic websites ke liye use hota hai. Abhi tak zyada tar internet IPv4 pe chal raha hai.

IPv6 modern systems aur zyada devices ke liye use hota hai (jaise IoT: Internet of Things). Faster, secure, aur future ke liye ready hai.

IPv4 (Internet Protocol Version 4):

Purana system hai jo internet devices ko unique address deta hai. 4 parts ka address hota hai, aur har part numbers (0-255) mein hota hai. Example: 192.168.1.1 Total 4.3 billion addresses provide kar sakta hai (limited hai). Dots (.) se separated hota hai. Devices zyada ho gaye hain, toh ab IPv4 addresses kam pad rahe hain.

IPv6 (Internet Protocol Version 6):

Naya system hai, jo zyada advanced aur bada hai. 8 parts ka address hota hai, aur har part hexadecimal (letters + numbers) hota hai. Example: 2001:0db8:85a3:0000:0000:8a2e:0370:7334 Total unlimited addresses provide kar sakta hai (practically endless). Colons (:) se separated hota hai. Faster, secure, aur modern internet ke liye better. Future-proof (kabhi khatam nahi hoga).

Ek Example:

Tum Google.com kholte ho Tumhara browser request bhejta hai (using DNS) aur uska IP address (IPv4 ya IPv6) leta hai. IP address server tak request pahuchata hai. Server data wapas tumhare IP address ko bhejta hai, aur tumhare screen par website dikhai deti hai.

Conclusion

IPv4: Purana, chhota, limited addresses. Example: 192.168.0.1

IPv6: Naya, bada, aur unlimited addresses. Example: 2001:0db8:85a3:0000:0000:8a2e:0370:7334
